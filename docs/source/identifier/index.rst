Identifiers:
************

Identifiers help to extract certain data from strickgraph.

All identifiers should follow a certain form, described by the abstract class
:py:class:`Identifier<createcloth.abstract_identifier.identifier>`. 
This abstract class is more a guidance, how the identifier should work, than 
a strict dictation.
