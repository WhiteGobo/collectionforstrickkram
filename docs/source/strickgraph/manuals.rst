Manuals for Strickgraph
-----------------------

Manual generation is implemented into the strickgraphclass via subclass 
:py:class:`strick_manualhelper<createcloth.strickgraph.strickgraph_toknitmanual.strick_manualhelper>`.
This subclass provides method for manual generation :py:meth:`to_manual<createcloth.strickgraph.strickgraph_toknitmanual.strick_manualhelper.to_manual>` 
and a generator :py:meth:`from_manual<createcloth.strickgraph.strickgraph_toknitmanual.strick_manualhelper.from_manual>` from a strickgraph.


Comment on classing
......................

As :py:class:`class_special_commands<createcloth.strickgraph.strickgraph_toknitmanual.strick_manualhelper>` 
provides a generator, dont alterate input of :py:meth:`__init__` of parentclasses.
